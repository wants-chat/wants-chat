## WantsChatとは？

WantsChatは、あなたの欲求を理解し、適切なUIを動的に生成するAIプラットフォームです。テキストで応答するだけの従来のチャットボットとは異なり、インテントを検出してコンテキストに合ったインターフェースを提供します。

## 主な機能

- **インテント検出** — 自然言語から意図を理解し、最適なアプリを起動
- **1100以上のAIアプリ** — コンバーター、ジェネレーター、分析、自動化ツール
- **アプリビルダー** — コードなしでカスタムAIアプリを作成
- **ワークフロー自動化** — マルチステップのAI自動化チェーン
- **ブラウザ拡張機能** — どのウェブサイトからでもアクセス
- **セルフホスト対応** — 自分のインフラで実行可能
