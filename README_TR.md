## 🎯 WantsChat Nedir?

**WantsChat**, ne istediğinizi anlayan ve bunu gerçekleştirmek için doğru aracı, uygulamayı veya otomasyonu anında sağlayan devrim niteliğinde, yapay zeka destekli bir platformdur. Yalnızca metinle yanıt veren geleneksel sohbet botlarının aksine, WantsChat niyetinizi algılar ve tam ihtiyaçlarınıza göre uyarlanmış **bağlamsal kullanıcı arayüzleri** oluşturur.

```
💬 Diyorsunuz ki: "500 USD'yi EUR'ya çevirmek istiyorum"
✨ WantsChat: Anında canlı kurlarla şık bir döviz çevirici arayüzü gösterir

💬 Diyorsunuz ki: "Girişimim için bir logo oluşturmak istiyorum"
✨ WantsChat: Önceden yüklenmiş logo şablonlarıyla AI görsel oluşturucusunu açar

💬 Diyorsunuz ki: "Proje giderlerimi takip etmek istiyorum"
✨ WantsChat: Para biriminiz, kategorileriniz ve dışa aktarma seçeneklerinizle gider takipçisi gösterir
```

### 🚀 Çözdüğümüz Sorun

2025'te ortalama bir bilgi çalışanı her gün **50'den fazla farklı uygulama** kullanıyor:
- Hesaplayıcılar, çeviriciler, jeneratörler
- Proje yönetim araçları
- Tasarım yazılımları
- Finans takipçileri
- Sağlık uygulamaları
- Ve onlarcası daha...

**WantsChat bunların TÜMÜNÜ tek bir akıllı platformda birleştirir.**

## 🛠️ Teknoloji Yığını

### Ön Uç
- **React 18** + TypeScript
- **Vite** derleme araçları
- **Tailwind CSS** + shadcn/ui
- Animasyonlar için **Framer Motion**
- Veri çekme için **TanStack Query**
- Uluslararasılaştırma için **i18next**

### Arka Uç
- **NestJS** (Node.js çerçevesi)
- **PostgreSQL** veritabanı (ham `pg` sürücüsü)
- **Qdrant** vektör veritabanı (isteğe bağlı)
- **Redis** (BullMQ kuyrukları aracılığıyla)
- Gerçek zamanlı iletişim için **Socket.io**
- API belgelemesi için **Swagger/OpenAPI**

### Tarayıcı Uzantısı
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- React bileşenlerini web uygulamasıyla kaynak düzeyinde içe aktarımlar yoluyla paylaşır

### AI/ML
- **OpenRouter** birleşik LLM ağ geçidi (30+ model)
- Görsel oluşturma için **Runware**
- Anlamsal arama için **OpenAI Embeddings**

### Altyapı
- **Docker** + Docker Compose
- **Cloudflare R2** depolama
- Herhangi bir bulut sağlayıcıda kendi sunucusunda barındırılabilir

## 🚀 Başlarken

En hızlı yol:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

Ardından `http://localhost:5173` adresini açın.

Ön koşullar, ortam değişkenleri, Docker'sız yol, isteğe bağlı özellikler ve sorun giderme için **[`DEVELOPMENT.md`](DEVELOPMENT.md)** belgesine bakın — bu, katkıda bulunan adaylarına yönelik resmi onboarding rehberidir.

---

## 🧩 Tarayıcı Uzantısı

Bir Manifest V3 tarayıcı uzantısı [`extension/`](extension/) dizininde bulunur ve WantsChat'i herhangi bir sekmeye getirir — vurgulanan metinlerde araçlar çağırın, parçacıkları kaydedin ve sayfadan ayrılmadan sohbet edin.

```bash
cd extension
npm install
npm run build
# Ardından extension/dist klasörünü Chrome/Edge/Firefox'ta paketlenmemiş uzantı olarak yükleyin
```
