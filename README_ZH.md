## 什么是WantsChat？

WantsChat是一个革命性的AI平台，理解你想要什么，并动态渲染适合的用户界面。1100+内置AI应用，支持自定义应用构建和工作流自动化。

## 主要功能

- **意图检测** — 从自然语言理解用户意图
- **1100+ AI应用** — 转换器、生成器、分析工具等
- **应用构建器** — 零代码创建自定义AI应用
- **工作流自动化** — 多步骤AI自动化链
- **浏览器扩展** — 任何网站上使用
- **自托管** — 可在自有服务器上运行
