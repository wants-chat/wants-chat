## 🎯 什么是 WantsChat?

**WantsChat** 是一个革命性的 AI 驱动平台,它能理解你想要什么,并立即提供合适的工具、应用或自动化来完成它。与只能用文字回复的传统聊天机器人不同,WantsChat 检测你的意图,并渲染**根据你的具体需求量身定制的上下文用户界面**。

```
💬 你说:"I want to convert 500 USD to EUR"
✨ WantsChat:立即显示带有实时汇率的精美货币转换器界面

💬 你说:"I want to generate a logo for my startup"
✨ WantsChat:打开预加载了徽标模板的 AI 图像生成器

💬 你说:"I want to track my project expenses"
✨ WantsChat:显示带有你的货币、类别和导出选项的费用追踪器
```

### 🚀 我们要解决的问题

到 2025 年,普通的知识工作者每天使用**超过 50 个不同的应用**:
- 计算器、转换器、生成器
- 项目管理工具
- 设计软件
- 财务追踪器
- 健康应用
- 还有数十种更多...

**WantsChat 将它们全部整合到一个智能平台中。**

## 🛠️ 技术栈

### 前端
- **React 18** + TypeScript
- **Vite** 构建工具
- **TailwindCSS** + shadcn/ui
- **Framer Motion** 用于动画
- **TanStack Query** 用于数据获取
- **i18next** 用于国际化

### 后端
- **NestJS** (Node.js 框架)
- **PostgreSQL** 数据库 (原始 `pg` 驱动)
- **Qdrant** 向量数据库 (可选)
- **Redis** (通过 BullMQ 队列)
- **Socket.io** 用于实时通信
- **Swagger/OpenAPI** 用于 API 文档

### 浏览器扩展
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome、Edge、Firefox)
- 通过源代码级导入与 web 应用共享 React 组件

### AI/ML
- **OpenRouter** 统一 LLM 网关 (30+ 模型)
- **Runware** 用于图像生成
- **OpenAI Embeddings** 用于语义搜索

### 基础设施
- **Docker** + Docker Compose
- **Cloudflare R2** 存储
- 可在任何云提供商上自托管

## 🚀 入门

最快路径:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

然后打开 `http://localhost:5173`。

有关先决条件、环境变量、不使用 Docker 的路径、可选功能和故障排除,请参阅 **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — 贡献者入门的权威指南。

---

## 🧩 浏览器扩展

[`extension/`](extension/) 目录中有一个 Manifest V3 浏览器扩展,可以将 WantsChat 带入任何标签页 — 在突出显示的文本上调用工具、保存代码片段,以及在不离开页面的情况下聊天。

```bash
cd extension
npm install
npm run build
# 然后在 Chrome/Edge/Firefox 中将 extension/dist 加载为未打包的扩展
```
