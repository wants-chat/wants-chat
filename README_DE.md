## 🎯 Was ist WantsChat?

**WantsChat** ist eine revolutionäre KI-gestützte Plattform, die versteht, was du willst, und sofort das passende Tool, die App oder die Automatisierung bereitstellt, um es zu erledigen. Anders als herkömmliche Chatbots, die nur mit Text antworten, erkennt WantsChat deine Absicht und rendert **kontextbezogene Benutzeroberflächen**, die genau auf deine Bedürfnisse zugeschnitten sind.

```
💬 Du sagst: "Ich möchte 500 USD in EUR umrechnen"
✨ WantsChat: Zeigt sofort eine schöne Währungsumrechner-Oberfläche mit Live-Kursen

💬 Du sagst: "Ich möchte ein Logo für mein Startup generieren"
✨ WantsChat: Öffnet den KI-Bildgenerator mit vorgeladenen Logo-Vorlagen

💬 Du sagst: "Ich möchte meine Projektausgaben verfolgen"
✨ WantsChat: Zeigt einen Ausgaben-Tracker mit deiner Währung, Kategorien und Export-Optionen
```

### 🚀 Das Problem, das wir lösen

Im Jahr 2025 nutzt der durchschnittliche Wissensarbeiter täglich **mehr als 50 verschiedene Apps**:
- Rechner, Konverter, Generatoren
- Projektmanagement-Tools
- Design-Software
- Finanz-Tracker
- Gesundheits-Apps
- Und Dutzende mehr...

**WantsChat kombiniert sie ALLE in einer intelligenten Plattform.**

## 🛠️ Tech-Stack

### Frontend
- **React 18** + TypeScript
- **Vite** als Build-Tool
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** für Animationen
- **TanStack Query** zum Datenabruf
- **i18next** für Internationalisierung

### Backend
- **NestJS** (Node.js-Framework)
- **PostgreSQL**-Datenbank (roher `pg`-Treiber)
- **Qdrant**-Vektordatenbank (optional)
- **Redis** (über BullMQ-Queues)
- **Socket.io** für Echtzeit-Kommunikation
- **Swagger/OpenAPI** für API-Dokumentation

### Browser-Erweiterung
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- Teilt React-Komponenten mit der Web-App über Quellcode-Imports

### KI/ML
- **OpenRouter** als einheitliches LLM-Gateway (30+ Modelle)
- **Runware** für Bildgenerierung
- **OpenAI Embeddings** für semantische Suche

### Infrastruktur
- **Docker** + Docker Compose
- **Cloudflare R2**-Speicher
- Selbst-hostbar bei jedem Cloud-Anbieter

## 🚀 Erste Schritte

Der schnellste Weg:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

Dann öffne `http://localhost:5173`.

Für Voraussetzungen, Umgebungsvariablen, den Pfad ohne Docker, optionale Funktionen und Fehlerbehebung siehe **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — den maßgeblichen Onboarding-Leitfaden für Mitwirkende.

---

## 🧩 Browser-Erweiterung

Eine Manifest-V3-Browser-Erweiterung liegt in [`extension/`](extension/) und bringt WantsChat in jeden Tab — rufe Tools auf markiertem Text auf, speichere Snippets und chatte, ohne die Seite zu verlassen.

```bash
cd extension
npm install
npm run build
# Lade dann extension/dist als entpackte Erweiterung in Chrome/Edge/Firefox
```
