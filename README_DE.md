## Was ist wants.chat?

KI-Plattform die Ihre Absichten versteht und kontextbezogene Benutzeroberflaechen generiert. 1100+ KI-Apps, App-Builder und Workflow-Automatisierung.

## Hauptfunktionen

- **Absichtserkennung** — Versteht Beduerfnisse aus natuerlicher Sprache
- **1100+ KI-Apps** — Konverter, Generatoren, Analysetools
- **App-Builder** — Erstellen Sie KI-Apps ohne Code
- **Automatisierung** — Mehrstufige KI-Automatisierungsketten
- **Browser-Erweiterung** — Zugriff von jeder Website
- **Selbst gehostet** — Auf eigener Infrastruktur betreiben
