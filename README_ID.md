## 🎯 Apa itu WantsChat?

**WantsChat** adalah platform berbasis AI revolusioner yang memahami apa yang Anda inginkan dan secara instan menyediakan alat, aplikasi, atau otomatisasi yang tepat untuk menyelesaikannya. Tidak seperti chatbot tradisional yang hanya merespons dengan teks, WantsChat mendeteksi niat Anda dan merender **antarmuka pengguna kontekstual** yang disesuaikan dengan kebutuhan spesifik Anda.

```
💬 Anda berkata: "Saya ingin mengonversi 500 USD ke EUR"
✨ WantsChat: Langsung menampilkan UI konverter mata uang yang indah dengan kurs langsung

💬 Anda berkata: "Saya ingin membuat logo untuk startup saya"
✨ WantsChat: Membuka generator gambar AI dengan template logo yang sudah dimuat

💬 Anda berkata: "Saya ingin melacak pengeluaran proyek saya"
✨ WantsChat: Menampilkan pelacak pengeluaran dengan mata uang, kategori, dan opsi ekspor Anda
```

### 🚀 Masalah yang Kami Pecahkan

Pada tahun 2025, rata-rata pekerja pengetahuan menggunakan **lebih dari 50 aplikasi berbeda** setiap hari:
- Kalkulator, konverter, generator
- Alat manajemen proyek
- Perangkat lunak desain
- Pelacak keuangan
- Aplikasi kesehatan
- Dan lusinan lainnya...

**WantsChat menggabungkan SEMUANYA ke dalam satu platform cerdas.**

## 🛠️ Tech Stack

### Frontend
- **React 18** + TypeScript
- Build tooling **Vite**
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** untuk animasi
- **TanStack Query** untuk pengambilan data
- **i18next** untuk internasionalisasi

### Backend
- **NestJS** (framework Node.js)
- Database **PostgreSQL** (driver `pg` mentah)
- Database vektor **Qdrant** (opsional)
- **Redis** (melalui antrian BullMQ)
- **Socket.io** untuk komunikasi real-time
- **Swagger/OpenAPI** untuk dokumentasi API

### Ekstensi Browser
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- Berbagi komponen React dengan aplikasi web melalui impor tingkat sumber

### AI/ML
- Gateway LLM terpadu **OpenRouter** (30+ model)
- **Runware** untuk pembuatan gambar
- **OpenAI Embeddings** untuk pencarian semantik

### Infrastruktur
- **Docker** + Docker Compose
- Penyimpanan **Cloudflare R2**
- Dapat di-host sendiri di penyedia cloud manapun

## 🚀 Memulai

Jalur tercepat:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

Lalu buka `http://localhost:5173`.

Untuk prasyarat, variabel env, jalur tanpa Docker, fitur opsional, dan pemecahan masalah, lihat **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — panduan onboarding kontributor kanonik.

---

## 🧩 Ekstensi Browser

Ekstensi browser Manifest V3 berada di [`extension/`](extension/) dan menghadirkan WantsChat ke tab manapun — panggil alat pada teks yang disorot, simpan cuplikan, dan obrol tanpa meninggalkan halaman.

```bash
cd extension
npm install
npm run build
# Lalu muat extension/dist sebagai ekstensi tidak terbungkus di Chrome/Edge/Firefox
```
