## 🎯 Cos'è WantsChat?

**WantsChat** è una piattaforma rivoluzionaria basata sull'IA che capisce ciò che vuoi e fornisce istantaneamente lo strumento, l'app o l'automazione giusti per portarlo a termine. A differenza dei chatbot tradizionali che rispondono solo con testo, WantsChat rileva la tua intenzione e visualizza **interfacce utente contestuali** su misura per le tue esigenze esatte.

```
💬 Tu dici: "Voglio convertire 500 USD in EUR"
✨ WantsChat: Mostra istantaneamente un bellissimo convertitore di valuta con tassi in tempo reale

💬 Tu dici: "Voglio generare un logo per la mia startup"
✨ WantsChat: Apre il generatore di immagini IA con modelli di logo precaricati

💬 Tu dici: "Voglio tenere traccia delle spese del mio progetto"
✨ WantsChat: Mostra un tracker delle spese con la tua valuta, categorie e opzioni di esportazione
```

### 🚀 Il problema che risolviamo

Nel 2025, il knowledge worker medio utilizza **più di 50 app** diverse al giorno:
- Calcolatori, convertitori, generatori
- Strumenti di gestione dei progetti
- Software di design
- Tracker finanziari
- App per la salute
- E decine di altri...

**WantsChat li combina TUTTI in un'unica piattaforma intelligente.**

## 🛠️ Stack tecnologico

### Frontend
- **React 18** + TypeScript
- **Vite** come strumento di build
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** per le animazioni
- **TanStack Query** per il recupero dei dati
- **i18next** per l'internazionalizzazione

### Backend
- **NestJS** (framework Node.js)
- Database **PostgreSQL** (driver `pg` raw)
- Database vettoriale **Qdrant** (opzionale)
- **Redis** (tramite code BullMQ)
- **Socket.io** per la comunicazione in tempo reale
- **Swagger/OpenAPI** per la documentazione API

### Estensione browser
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- Condivide componenti React con la web app tramite import a livello sorgente

### IA/ML
- Gateway LLM unificato **OpenRouter** (più di 30 modelli)
- **Runware** per la generazione di immagini
- **OpenAI Embeddings** per la ricerca semantica

### Infrastruttura
- **Docker** + Docker Compose
- Storage **Cloudflare R2**
- Self-hostable su qualsiasi provider cloud

## 🚀 Per iniziare

Il percorso più rapido:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

Quindi apri `http://localhost:5173`.

Per i prerequisiti, le variabili d'ambiente, il percorso senza Docker, le funzionalità opzionali e la risoluzione dei problemi, consulta **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — la guida canonica all'onboarding dei contributori.

---

## 🧩 Estensione browser

Un'estensione browser Manifest V3 si trova in [`extension/`](extension/) e porta WantsChat in qualsiasi tab — invoca strumenti su testo evidenziato, salva snippet e chatta senza lasciare la pagina.

```bash
cd extension
npm install
npm run build
# Quindi carica extension/dist come estensione non pacchettizzata in Chrome/Edge/Firefox
```
