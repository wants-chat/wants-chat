## 🎯 O que é WantsChat?

**WantsChat** é uma plataforma revolucionária baseada em IA que entende o que você quer e instantaneamente fornece a ferramenta, aplicativo ou automação certa para realizar a tarefa. Diferente dos chatbots tradicionais que respondem apenas com texto, o WantsChat detecta sua intenção e renderiza **interfaces de usuário contextuais** adaptadas às suas necessidades exatas.

```
💬 Você diz: "Quero converter 500 USD para EUR"
✨ WantsChat: Mostra instantaneamente uma bela interface de conversor de moedas com taxas em tempo real

💬 Você diz: "Quero gerar um logotipo para minha startup"
✨ WantsChat: Abre o gerador de imagens IA com modelos de logotipo pré-carregados

💬 Você diz: "Quero acompanhar as despesas do meu projeto"
✨ WantsChat: Exibe o rastreador de despesas com sua moeda, categorias e opções de exportação
```

### 🚀 O Problema que Resolvemos

Em 2025, o trabalhador do conhecimento médio usa **mais de 50 aplicativos diferentes** diariamente:
- Calculadoras, conversores, geradores
- Ferramentas de gerenciamento de projetos
- Software de design
- Rastreadores financeiros
- Aplicativos de saúde
- E dezenas mais...

**O WantsChat combina TODOS eles em uma única plataforma inteligente.**

## 🛠️ Stack Tecnológica

### Frontend
- **React 18** + TypeScript
- Ferramentas de build **Vite**
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** para animações
- **TanStack Query** para busca de dados
- **i18next** para internacionalização

### Backend
- **NestJS** (framework Node.js)
- Banco de dados **PostgreSQL** (driver `pg` nativo)
- Banco de dados vetorial **Qdrant** (opcional)
- **Redis** (via filas BullMQ)
- **Socket.io** para comunicação em tempo real
- **Swagger/OpenAPI** para documentação da API

### Extensão para Navegador
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- Compartilha componentes React com a aplicação web via importações de código-fonte

### IA/ML
- Gateway unificado de LLM **OpenRouter** (mais de 30 modelos)
- **Runware** para geração de imagens
- **OpenAI Embeddings** para busca semântica

### Infraestrutura
- **Docker** + Docker Compose
- Armazenamento **Cloudflare R2**
- Auto-hospedável em qualquer provedor de nuvem

## 🚀 Primeiros Passos

O caminho mais rápido:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

Em seguida, abra `http://localhost:5173`.

Para pré-requisitos, variáveis de ambiente, o caminho sem Docker, recursos opcionais e solução de problemas, consulte **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — o guia canônico de onboarding para contribuidores.

---

## 🧩 Extensão para Navegador

Uma extensão de navegador Manifest V3 vive em [`extension/`](extension/) e leva o WantsChat para qualquer aba — invoque ferramentas em texto destacado, salve trechos e converse sem sair da página.

```bash
cd extension
npm install
npm run build
# Em seguida, carregue extension/dist como uma extensão descompactada no Chrome/Edge/Firefox
```
