## 🎯 Qu'est-ce que WantsChat ?

**WantsChat** est une plateforme révolutionnaire propulsée par l'IA qui comprend ce que vous voulez et fournit instantanément le bon outil, l'application ou l'automatisation pour le réaliser. Contrairement aux chatbots traditionnels qui ne répondent que par du texte, WantsChat détecte votre intention et affiche des **interfaces utilisateur contextuelles** adaptées à vos besoins exacts.

```
💬 Vous dites : "Je veux convertir 500 USD en EUR"
✨ WantsChat : Affiche instantanément un magnifique convertisseur de devises avec des taux en direct

💬 Vous dites : "Je veux générer un logo pour ma startup"
✨ WantsChat : Ouvre le générateur d'images IA avec des modèles de logos préchargés

💬 Vous dites : "Je veux suivre les dépenses de mon projet"
✨ WantsChat : Affiche un suivi des dépenses avec votre devise, vos catégories et vos options d'export
```

### 🚀 Le problème que nous résolvons

En 2025, le travailleur du savoir moyen utilise **plus de 50 applications** différentes par jour :
- Calculatrices, convertisseurs, générateurs
- Outils de gestion de projet
- Logiciels de design
- Suivis financiers
- Applications de santé
- Et des dizaines d'autres...

**WantsChat les combine TOUS en une seule plateforme intelligente.**

## 🛠️ Stack technique

### Frontend
- **React 18** + TypeScript
- **Vite** comme outil de build
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** pour les animations
- **TanStack Query** pour la récupération de données
- **i18next** pour l'internationalisation

### Backend
- **NestJS** (framework Node.js)
- Base de données **PostgreSQL** (driver `pg` brut)
- Base de données vectorielle **Qdrant** (optionnelle)
- **Redis** (via les files BullMQ)
- **Socket.io** pour la communication temps réel
- **Swagger/OpenAPI** pour la documentation de l'API

### Extension de navigateur
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- Partage les composants React avec l'app web via des imports au niveau source

### IA/ML
- Passerelle LLM unifiée **OpenRouter** (30+ modèles)
- **Runware** pour la génération d'images
- **OpenAI Embeddings** pour la recherche sémantique

### Infrastructure
- **Docker** + Docker Compose
- Stockage **Cloudflare R2**
- Auto-hébergeable sur n'importe quel fournisseur cloud

## 🚀 Pour commencer

Le chemin le plus rapide :

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

Puis ouvrez `http://localhost:5173`.

Pour les prérequis, les variables d'environnement, le chemin sans Docker, les fonctionnalités optionnelles et le dépannage, consultez **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — le guide canonique d'onboarding des contributeurs.

---

## 🧩 Extension de navigateur

Une extension de navigateur Manifest V3 se trouve dans [`extension/`](extension/) et amène WantsChat dans n'importe quel onglet — invoquez des outils sur du texte surligné, sauvegardez des extraits et discutez sans quitter la page.

```bash
cd extension
npm install
npm run build
# Puis chargez extension/dist comme une extension non empaquetée dans Chrome/Edge/Firefox
```
