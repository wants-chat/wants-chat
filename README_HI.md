## 🎯 WantsChat क्या है?

**WantsChat** एक क्रांतिकारी AI-संचालित प्लेटफ़ॉर्म है जो समझता है कि आप क्या चाहते हैं और तुरंत उसे पूरा करने के लिए सही टूल, ऐप या ऑटोमेशन (automation) प्रदान करता है। पारंपरिक चैटबॉट्स के विपरीत जो केवल टेक्स्ट से जवाब देते हैं, WantsChat आपके इरादे (intent) का पता लगाता है और आपकी सटीक ज़रूरतों के अनुरूप **कॉन्टेक्स्चुअल यूज़र इंटरफ़ेस (contextual user interfaces)** रेंडर करता है।

```
💬 आप कहते हैं: "I want to convert 500 USD to EUR"
✨ WantsChat: तुरंत लाइव रेट्स के साथ एक सुंदर करेंसी कन्वर्टर UI दिखाता है

💬 आप कहते हैं: "I want to generate a logo for my startup"
✨ WantsChat: पहले से लोड किए गए लोगो टेम्पलेट्स के साथ AI इमेज जनरेटर खोलता है

💬 आप कहते हैं: "I want to track my project expenses"
✨ WantsChat: आपकी मुद्रा, श्रेणियाँ और एक्सपोर्ट विकल्पों के साथ खर्च ट्रैकर दिखाता है
```

### 🚀 हम जो समस्या हल करते हैं

2025 में, औसत ज्ञान-कर्मी (knowledge worker) रोज़ाना **50 से अधिक विभिन्न ऐप्स** का उपयोग करता है:
- कैलकुलेटर, कन्वर्टर, जनरेटर
- प्रोजेक्ट प्रबंधन उपकरण

- वित्तीय ट्रैकर
- स्वास्थ्य ऐप्स
- और दर्जनों अन्य...

**WantsChat इन सभी को एक बुद्धिमान प्लेटफ़ॉर्म में संयोजित करता है।**

### बैकएंड

- **PostgreSQL** डेटाबेस (raw `pg` ड्राइवर)
- **Qdrant** वेक्टर डेटाबेस (वैकल्पिक)
- **Redis** (BullMQ क्यू के माध्यम से)
- **Socket.io** रीयल-टाइम संचार के लिए
- **Swagger/OpenAPI** API दस्तावेज़ीकरण के लिए

### ब्राउज़र एक्सटेंशन
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- सोर्स-स्तरीय आयातों के माध्यम से वेब ऐप के साथ React कंपोनेंट साझा करता है

### AI/ML
- **OpenRouter** एकीकृत LLM गेटवे (30+ मॉडल)
- **Runware** इमेज जनरेशन के लिए
- **OpenAI Embeddings** सेमांटिक सर्च के लिए

### इन्फ्रास्ट्रक्चर
- **Docker** + Docker Compose
- **Cloudflare R2** स्टोरेज
- किसी भी क्लाउड प्रदाता पर सेल्फ़-होस्टेबल

## 🚀 शुरू कैसे करें

सबसे तेज़ रास्ता:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

फिर `http://localhost:5173` खोलें।

पूर्वापेक्षाओं, env variables, बिना-Docker पथ, वैकल्पिक सुविधाओं और समस्या निवारण के लिए, **[`DEVELOPMENT.md`](DEVELOPMENT.md)** देखें — योगदानकर्ता ऑनबोर्डिंग की आधिकारिक गाइड।

---

## 🧩 ब्राउज़र एक्सटेंशन

एक Manifest V3 ब्राउज़र एक्सटेंशन [`extension/`](extension/) में रहता है और WantsChat को किसी भी टैब में लाता है — हाइलाइट किए गए टेक्स्ट पर टूल्स को आमंत्रित करें, स्निपेट सहेजें, और पेज छोड़े बिना चैट करें।

```bash
cd extension
npm install
npm run build
# फिर Chrome/Edge/Firefox में extension/dist को unpacked extension के रूप में लोड करें
```
