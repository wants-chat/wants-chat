## 🎯 Что такое WantsChat?

**WantsChat** — это революционная платформа на основе ИИ, которая понимает, что вы хотите, и мгновенно предоставляет нужный инструмент, приложение или автоматизацию для выполнения задачи. В отличие от традиционных чат-ботов, которые отвечают только текстом, WantsChat определяет ваше намерение и отображает **контекстные пользовательские интерфейсы**, адаптированные под ваши конкретные нужды.

```
💬 Вы говорите: "Хочу конвертировать 500 USD в EUR"
✨ WantsChat: Мгновенно показывает красивый интерфейс конвертера валют с актуальными курсами

💬 Вы говорите: "Хочу создать логотип для моего стартапа"
✨ WantsChat: Открывает ИИ-генератор изображений с предзагруженными шаблонами логотипов

💬 Вы говорите: "Хочу отслеживать расходы на проект"
✨ WantsChat: Отображает трекер расходов с вашей валютой, категориями и опциями экспорта
```

### 🚀 Какую проблему мы решаем

В 2025 году среднестатистический работник умственного труда использует **более 50 различных приложений** ежедневно:
- Калькуляторы, конвертеры, генераторы
- Инструменты управления проектами
- Программы для дизайна
- Финансовые трекеры
- Приложения для здоровья
- И десятки других...

**WantsChat объединяет их ВСЕ в одной интеллектуальной платформе.**

## 🛠️ Технологический стек

### Frontend
- **React 18** + TypeScript
- **Vite** для сборки
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** для анимаций
- **TanStack Query** для загрузки данных
- **i18next** для интернационализации

### Backend
- **NestJS** (Node.js фреймворк)
- База данных **PostgreSQL** (нативный драйвер `pg`)
- Векторная база **Qdrant** (опционально)
- **Redis** (через очереди BullMQ)
- **Socket.io** для коммуникации в реальном времени
- **Swagger/OpenAPI** для документации API

### Расширение для браузера
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- Делит React-компоненты с веб-приложением через импорты на уровне исходного кода

### AI/ML
- Унифицированный шлюз LLM **OpenRouter** (30+ моделей)
- **Runware** для генерации изображений
- **OpenAI Embeddings** для семантического поиска

### Инфраструктура
- **Docker** + Docker Compose
- Хранилище **Cloudflare R2**
- Возможность самостоятельного хостинга у любого облачного провайдера

## 🚀 Быстрый старт

Самый быстрый путь:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

Затем откройте `http://localhost:5173`.

Для предварительных требований, переменных окружения, пути без Docker, дополнительных функций и устранения неполадок см. **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — каноническое руководство по онбордингу контрибьюторов.

---

## 🧩 Расширение для браузера

Расширение браузера на Manifest V3 находится в [`extension/`](extension/) и приносит WantsChat в любую вкладку — вызывайте инструменты для выделенного текста, сохраняйте сниппеты и общайтесь, не покидая страницы.

```bash
cd extension
npm install
npm run build
# Затем загрузите extension/dist как распакованное расширение в Chrome/Edge/Firefox
```
