## Что такое wants.chat?

ИИ-платформа, которая понимает ваши намерения и генерирует контекстные интерфейсы. 1100+ ИИ-приложений, конструктор приложений и автоматизация рабочих процессов.

## Основные возможности

- **Определение намерений** — Понимает потребности на естественном языке
- **1100+ ИИ-приложений** — Конвертеры, генераторы, инструменты анализа
- **Конструктор приложений** — Создавайте ИИ-приложения без кода
- **Автоматизация** — Многоэтапные цепочки ИИ-автоматизации
- **Расширение для браузера** — Доступ с любого сайта
- **Самостоятельный хостинг** — Запуск на собственной инфраструктуре

Подробности в [английском README](./README.md).
