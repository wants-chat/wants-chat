## 🎯 Wat is WantsChat?

**WantsChat** is een revolutionair AI-platform dat begrijpt wat je wilt en je direct het juiste hulpmiddel, de juiste app of automatisering biedt om de klus te klaren. In tegenstelling tot traditionele chatbots die alleen tekst antwoorden, detecteert WantsChat je intentie en rendert **contextuele gebruikersinterfaces** die zijn afgestemd op jouw exacte behoeften.

```
💬 Jij zegt: "Ik wil 500 USD naar EUR converteren"
✨ WantsChat: Toont onmiddellijk een prachtige valutaconverter-UI met live koersen

💬 Jij zegt: "Ik wil een logo voor mijn startup maken"
✨ WantsChat: Opent AI-beeldgenerator met logosjablonen vooraf geladen

💬 Jij zegt: "Ik wil mijn projectkosten bijhouden"
✨ WantsChat: Toont een uitgaventracker met jouw valuta, categorieën en exportopties
```

### 🚀 Het probleem dat we oplossen

In 2025 gebruikt de gemiddelde kenniswerker dagelijks **meer dan 50 verschillende apps**:
- Calculators, converters, generators
- Projectmanagementtools
- Designsoftware
- Financiële trackers
- Gezondheidsapps
- En nog tientallen meer...

**WantsChat combineert ze ALLEMAAL in één intelligent platform.**

## 🛠️ Tech Stack

### Frontend
- **React 18** + TypeScript
- **Vite** build-tooling
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** voor animaties
- **TanStack Query** voor data fetching
- **i18next** voor internationalisatie

### Backend
- **NestJS** (Node.js framework)
- **PostgreSQL** database (raw `pg` driver)
- **Qdrant** vectordatabase (optioneel)
- **Redis** (via BullMQ-wachtrijen)
- **Socket.io** voor realtime communicatie
- **Swagger/OpenAPI** voor API-documentatie

### Browserextensie
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- Deelt React-componenten met de webapp via imports op broncodeniveau

### AI/ML
- **OpenRouter** unified LLM-gateway (30+ modellen)
- **Runware** voor beeldgeneratie
- **OpenAI Embeddings** voor semantisch zoeken

### Infrastructuur
- **Docker** + Docker Compose
- **Cloudflare R2** opslag
- Zelf te hosten bij elke cloudprovider

## 🚀 Aan de slag

De snelste route:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

Open vervolgens `http://localhost:5173`.

Voor vereisten, env-variabelen, het pad zonder Docker, optionele functies en probleemoplossing zie **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — de canonieke onboardingsgids voor bijdragers.

---

## 🧩 Browserextensie

Een Manifest V3-browserextensie staat in [`extension/`](extension/) en brengt WantsChat naar elk tabblad — roep tools aan op gemarkeerde tekst, sla snippets op en chat zonder de pagina te verlaten.

```bash
cd extension
npm install
npm run build
# Laad daarna extension/dist als unpacked extension in Chrome/Edge/Firefox
```
