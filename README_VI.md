## 🎯 WantsChat là gì?

**WantsChat** là một nền tảng cách mạng được hỗ trợ bởi AI, hiểu được những gì bạn muốn và ngay lập tức cung cấp công cụ, ứng dụng hoặc tự động hóa phù hợp để hoàn thành điều đó. Khác với các chatbot truyền thống chỉ phản hồi bằng văn bản, WantsChat phát hiện ý định của bạn và hiển thị **giao diện người dùng theo ngữ cảnh** được thiết kế riêng cho nhu cầu chính xác của bạn.

```
💬 Bạn nói: "Tôi muốn chuyển đổi 500 USD sang EUR"
✨ WantsChat: Ngay lập tức hiển thị giao diện chuyển đổi tiền tệ đẹp mắt với tỷ giá trực tiếp

💬 Bạn nói: "Tôi muốn tạo logo cho startup của mình"
✨ WantsChat: Mở trình tạo hình ảnh AI với các mẫu logo được tải sẵn

💬 Bạn nói: "Tôi muốn theo dõi chi phí dự án của mình"
✨ WantsChat: Hiển thị trình theo dõi chi phí với tiền tệ, danh mục và các tùy chọn xuất của bạn
```

## 🛠️ Ngăn xếp công nghệ

### Frontend
- **React 18** + TypeScript
- Công cụ build **Vite**
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** cho hoạt ảnh
- **TanStack Query** để lấy dữ liệu
- **i18next** để quốc tế hóa

### Backend
- **NestJS** (framework Node.js)
- Cơ sở dữ liệu **PostgreSQL** (driver `pg` thô)
- Cơ sở dữ liệu vector **Qdrant** (tùy chọn)
- **Redis** (qua hàng đợi BullMQ)
- **Socket.io** cho giao tiếp thời gian thực
- **Swagger/OpenAPI** cho tài liệu API

### Tiện ích mở rộng trình duyệt
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- Chia sẻ các component React với ứng dụng web qua import cấp mã nguồn

### AI/ML
- Cổng LLM thống nhất **OpenRouter** (hơn 30 mô hình)
- **Runware** để tạo hình ảnh
- **OpenAI Embeddings** cho tìm kiếm ngữ nghĩa

### Cơ sở hạ tầng
- **Docker** + Docker Compose
- Lưu trữ **Cloudflare R2**
- Có thể tự lưu trữ trên bất kỳ nhà cung cấp đám mây nào

## 🚀 Bắt đầu

Cách nhanh nhất:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

Sau đó mở `http://localhost:5173`.

Để biết các điều kiện tiên quyết, biến môi trường, cách không dùng Docker, các tính năng tùy chọn và khắc phục sự cố, hãy xem **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — hướng dẫn onboarding chính thức cho cộng tác viên.

---

## 🧩 Tiện ích mở rộng trình duyệt

Tiện ích mở rộng trình duyệt Manifest V3 nằm trong [`extension/`](extension/) và mang WantsChat vào bất kỳ tab nào — gọi các công cụ trên văn bản được đánh dấu, lưu các đoạn trích và trò chuyện mà không cần rời khỏi trang.

```bash
cd extension
npm install
npm run build
# Sau đó tải extension/dist như một tiện ích mở rộng chưa đóng gói trong Chrome/Edge/Firefox
```

## 📄 Giấy phép

Dự án này được cấp phép theo **Giấy phép AGPL-3.0** - xem tệp [LICENSE](LICENSE) để biết chi tiết.

Điều này có nghĩa là bạn có thể tự do sử dụng, sửa đổi và phân phối phần mềm này, nhưng mọi sửa đổi cũng phải được mở mã nguồn theo cùng một giấy phép.
