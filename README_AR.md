## 🎯 ما هو WantsChat؟

**WantsChat** منصة ثورية مدعومة بالذكاء الاصطناعي تفهم ما تريده وتوفر لك على الفور الأداة أو التطبيق أو الأتمتة المناسبة لإنجازه. على عكس روبوتات الدردشة التقليدية التي تكتفي بالردود النصية، يكتشف WantsChat نيتك ويعرض **واجهات مستخدم سياقية** مصمَّمة خصيصًا لاحتياجاتك.

```
💬 تقول: "أريد تحويل 500 دولار أمريكي إلى يورو"
✨ WantsChat: يعرض على الفور واجهة محول عملات أنيقة بأسعار حية

💬 تقول: "أريد إنشاء شعار لشركتي الناشئة"
✨ WantsChat: يفتح مولِّد الصور بالذكاء الاصطناعي مع قوالب شعارات جاهزة

💬 تقول: "أريد تتبع نفقات مشروعي"
✨ WantsChat: يعرض متعقب نفقات بعملتك وفئاتك وخيارات تصدير
```

### 🚀 المشكلة التي نحلها

في عام 2025، يستخدم العامل المعرفي العادي **أكثر من 50 تطبيقًا مختلفًا** يوميًا:
- آلات حاسبة ومحوّلات ومولِّدات
- أدوات إدارة المشاريع
- برامج التصميم
- متعقبات مالية
- تطبيقات صحية
- وعشرات أخرى...

**يجمع WantsChat كل هذا في منصة ذكية واحدة.**

## 🛠️ المكدس التقني

### الواجهة الأمامية
- **React 18** + TypeScript
- أدوات بناء **Vite**
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** للرسوم المتحركة
- **TanStack Query** لجلب البيانات
- **i18next** للتدويل

### الخلفية
- **NestJS** (إطار عمل Node.js)
- قاعدة بيانات **PostgreSQL** (سائق `pg` الخام)
- قاعدة بيانات متجهية **Qdrant** (اختياري)
- **Redis** (عبر طوابير BullMQ)
- **Socket.io** للتواصل في الوقت الفعلي
- **Swagger/OpenAPI** لتوثيق واجهة API

### إضافة المتصفح
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome، Edge، Firefox)
- تشارك مكوّنات React مع تطبيق الويب عبر استيراد على مستوى المصدر

### الذكاء الاصطناعي/تعلم الآلة
- بوابة **OpenRouter** الموحدة لنماذج اللغة الكبيرة (أكثر من 30 نموذجًا)
- **Runware** لتوليد الصور
- **OpenAI Embeddings** للبحث الدلالي

### البنية التحتية
- **Docker** + Docker Compose
- تخزين **Cloudflare R2**
- قابل للاستضافة الذاتية على أي مزوّد سحابي

## 🚀 البدء

أسرع طريق:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

ثم افتح `http://localhost:5173`.

للمتطلبات الأساسية ومتغيرات البيئة والمسار بدون Docker والميزات الاختيارية واستكشاف الأخطاء، راجع **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — الدليل المعتمد لإعداد المساهمين.

---

## 🧩 إضافة المتصفح

تقع إضافة متصفح Manifest V3 في [`extension/`](extension/) وتجلب WantsChat إلى أي علامة تبويب — استدعِ الأدوات على النص المحدد، واحفظ المقتطفات، وتحدّث دون مغادرة الصفحة.

```bash
cd extension
npm install
npm run build
# ثم حمّل extension/dist كإضافة غير محزّمة في Chrome/Edge/Firefox
```
