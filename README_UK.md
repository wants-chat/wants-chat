## 🎯 Що таке WantsChat?

**WantsChat** — це революційна платформа на основі ШІ, яка розуміє, чого ви хочете, і миттєво надає потрібний інструмент, застосунок або автоматизацію для виконання завдання. На відміну від традиційних чат-ботів, які відповідають лише текстом, WantsChat визначає ваш намір і відображає **контекстні користувацькі інтерфейси**, адаптовані під ваші конкретні потреби.

```
💬 Ви кажете: "Хочу конвертувати 500 USD в EUR"
✨ WantsChat: Миттєво показує гарний інтерфейс конвертера валют з актуальними курсами

💬 Ви кажете: "Хочу створити логотип для мого стартапу"
✨ WantsChat: Відкриває ШІ-генератор зображень з попередньо завантаженими шаблонами логотипів

💬 Ви кажете: "Хочу відстежувати витрати проєкту"
✨ WantsChat: Показує трекер витрат з вашою валютою, категоріями та опціями експорту
```

### 🚀 Яку проблему ми вирішуємо

У 2025 році середньостатистичний працівник розумової праці використовує **понад 50 різних застосунків** щодня:
- Калькулятори, конвертери, генератори
- Інструменти управління проєктами
- Програми для дизайну
- Фінансові трекери
- Застосунки для здоров'я
- І десятки інших...

**WantsChat поєднує їх УСІ в одній інтелектуальній платформі.**

- **Розгортання**: Розгортання в один клік


## 🛠️ Технологічний стек

### Frontend
- **React 18** + TypeScript
- **Vite** для збірки
- **Tailwind CSS** + shadcn/ui
- **Framer Motion** для анімацій
- **TanStack Query** для отримання даних
- **i18next** для інтернаціоналізації

### Backend
- **NestJS** (Node.js фреймворк)
- База даних **PostgreSQL** (нативний драйвер `pg`)
- Векторна база **Qdrant** (опціонально)
- **Redis** (через черги BullMQ)
- **Socket.io** для комунікації в реальному часі
- **Swagger/OpenAPI** для документації API

### Розширення для браузера
- **Vite** + **TypeScript**
- **Manifest V3** (Chrome, Edge, Firefox)
- Ділить React-компоненти з вебзастосунком через імпорти на рівні вихідного коду

### AI/ML
- Уніфікований шлюз LLM **OpenRouter** (30+ моделей)
- **Runware** для генерації зображень
- **OpenAI Embeddings** для семантичного пошуку

### Інфраструктура
- **Docker** + Docker Compose
- Сховище **Cloudflare R2**
- Можливість самостійного хостингу в будь-якого хмарного провайдера

## 🚀 Швидкий старт

Найшвидший шлях:

```bash
git clone https://github.com/wants-chat/wants-chat.git
cd wants-chat
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
docker compose up
```

Потім відкрийте `http://localhost:5173`.

Для попередніх вимог, змінних середовища, шляху без Docker, додаткових функцій та усунення несправностей див. **[`DEVELOPMENT.md`](DEVELOPMENT.md)** — канонічний посібник з онбордингу контриб'юторів.

---

## 🧩 Розширення для браузера

Розширення браузера на Manifest V3 знаходиться в [`extension/`](extension/) і приносить WantsChat у будь-яку вкладку — викликайте інструменти для виділеного тексту, зберігайте сніпети та спілкуйтеся, не залишаючи сторінки.

```bash
cd extension
npm install
npm run build
# Потім завантажте extension/dist як розпаковане розширення в Chrome/Edge/Firefox
```
